# Changelog

此檔案記錄各版本的重要變更。

## 0.1.2

- 將最低系統需求下調至 macOS 15 Sequoia，同時保留 Apple 晶片與 Intel Mac 的通用版本。
- 新增 macOS 15 與 Xcode 26.3 的 row-action 相容性 CI，覆蓋滑動操作、釘選重排與捲動生命週期。
- 更新產品說明、安裝方式、授權與自動化驗證文件。

## 0.1.1

- 修正圖片剪輯釘選後未立即顯示釘選狀態與黃色指示線的問題。
- 改善 Homebrew 發布腳本對非預期輸出路徑值的錯誤處理。
- 更新 README 的 Mac App Store 與 Homebrew 安裝方式及正式連結。
