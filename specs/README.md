# Specification Index

> Authoritative index for all SDD specifications. Status vocabulary: `draft`, `active`, `blocked`, `completed`, `deprecated`, `superseded`, `cancelled`. In-progress specs (`draft`, `active`, `blocked`) live directly under `specs/` at the repository root; completed-and-accepted specs live in `specs/archive/YYYY/`; rejected/superseded/cancelled specs live in `specs/deprecated/`. There is no `specs/active/` directory. Do not delete historical specs.
