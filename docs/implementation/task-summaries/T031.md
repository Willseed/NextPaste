## Scope

T031 is the manual accessibility verification checklist for the 022 feature set. It covers VoiceOver navigation, mouse-only operation, system Accessibility settings (Increase Contrast, Reduce Transparency, Reduce Motion, Light/Dark/System appearance, Voice Control, Switch Control), and reporting of items that cannot be automatically verified.

## UITestAppLauncher polling review

`prepareMainWindow` was refactored to a deadline-bounded loop:

- Clear deadline: `let deadline = Date().addingTimeInterval(timeout)` with `while Date() < deadline`.
- No `sleep`, `Task.sleep`, or `asyncAfter`; uses XCTest `waitForExistence(timeout:)`.
- Not an unconditional retry — the loop exits when the ready button appears or the deadline passes.
- Retained because all UI tests depend on main-window readiness, including the T031 targeted test.
