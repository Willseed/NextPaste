# NextPaste

NextPaste 是一款為 macOS 打造的剪貼簿管理工具，會自動整理你複製過的文字與圖片，讓重要內容更容易搜尋、保存與再次使用。

## 為什麼需要 NextPaste？

剪貼簿通常只能保留最後一次複製的內容。工作做到一半，只要再複製另一段文字、連結或圖片，前一項就會被覆蓋；想找回時，只能回到原本的網頁、文件或對話重新翻找。

NextPaste 特別適合這些情境：

- **剛才複製的內容被蓋掉了**：自動保留剪貼簿歷史，不必反覆回到來源尋找。
- **經常重複使用相同資料**：釘選常用文字、連結或圖片，需要時立即取用。
- **在大量內容中找不到目標**：透過搜尋與分類篩選，快速縮小範圍。
- **需要抄下截圖裡的文字**：直接辨識並複製圖片中的文字，不必逐字輸入。
- **擔心剪貼簿內容被上傳**：所有歷史都留在這台 Mac，離線也能使用。

## 功能

- **自動保存剪貼簿歷史**：在 NextPaste 執行期間，自動記錄文字、圖片與螢幕截圖；切換到其他 App 或將視窗最小化後仍會持續運作。
- **減少重複與雜訊**：自動忽略空白文字及已保存的相同內容，也能手動建立文字剪輯。
- **快速搜尋與篩選**：即時搜尋歷史內容，並依全部、文字、圖片、已釘選或未釘選快速篩選。
- **常用內容隨手取用**：釘選重要項目、將文字或原始圖片複製回剪貼簿，或刪除不再需要的內容。
- **複製圖片中的文字**：直接辨識圖片內的文字並複製，省去手動輸入。
- **自訂全域快捷鍵**：不必離開目前使用的 App，就能快速叫出 NextPaste。
- **控制歷史容量**：設定保留 1–1000 筆未釘選內容；超過上限時會優先移除較舊項目，釘選內容不受影響。也可以只清除未釘選內容，或清除全部歷史。
- **資料留在本機**：剪貼簿歷史只保存在這台 Mac，不會傳送到伺服器，也不需要網路連線即可使用。
- **符合你的使用習慣**：支援繁體中文與英文，並可選擇跟隨系統、淺色或深色外觀。

## 系統需求

- macOS 26 Tahoe 或更新版本
- Apple 晶片或 Intel Mac

## 安裝

### Homebrew

在「終端機」執行：

```bash
brew install --cask willseed/tap/nextpaste
```

安裝完成後，即可從「應用程式」資料夾開啟 NextPaste。

### 手動安裝

1. 前往 [最新版本下載頁面](https://github.com/Willseed/NextPaste/releases/latest)。
2. 下載 `NextPaste-<版本號>.zip`。
3. 解壓縮後，將 `NextPaste.app` 移到「應用程式」資料夾。
4. 開啟 NextPaste，即可開始記錄剪貼簿內容。
